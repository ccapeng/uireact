All Communities come with a Shortcut navigation Menu which provides users with the quickest path from their current location to the most commonly used tools, apps, or pages. The Full Menu provides access to a complete list of all tools that have been made available (in Navigation Manager).

Select a community to customize a default shortcut menu (for specific roles) to the most commonly used tools for the members of that environment.

Shortcut menus will display by default to all users in each community. If you want to allow the community Host, Admin, or Faculty to make adjustments to the campus-default or set up their own shortcut menu, check "allow community customization" in the "More" button.
